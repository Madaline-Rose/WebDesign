# Retro Navigation Component

## Inspiration
This component is inspired by retro operating systems like Windows 95. 
I wanted to recreate the feeling of a draggable desktop window and 
turn it into a navigation bar.

## Features
- Flexbox layout for structure
- Window-style design with title bar
- Vertical navigation links
- Hover interactions

## What I Learned
- How to use Flexbox for both horizontal and vertical layouts
- How to structure components using nested containers
- How styling choices affect visual hierarchy

My inspo is from Pintrest and can be found in my images folder under the name Inspo.
