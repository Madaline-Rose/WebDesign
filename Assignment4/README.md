Working with Flexbox and Grid helped me better understand how modern websites organize content. I liked how Flexbox made it easy to align and center elements, especially for things like navigation bars and content previews. Grid was useful for creating structured layouts, such as image galleries, because it allows you to control rows and columns more precisely. So far, I slightly prefer Grid because it makes it easier to build organized layouts, but Flexbox is still very helpful for smaller sections of a page. Both tools will make future projects faster and more flexible because they reduce the need for complicated positioning or spacing tricks (sometjhing I struggled with in the past).
