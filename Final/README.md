
#Reflection

##What’s something new you learned while building your site
While building this site, I learned how to use CSS Grid in a more intentional way to create structured layouts instead of just stacking elements. In my gallery page, I used a 3-column by 2-row grid that alternates between text blocks and images, which helped me understand how layout design can guide visual flow. I also improved my understanding of how to organize a multi-page website using consistent navigation and shared styling across pages.

##How did you balance creativity with usability?
I balanced creativity with usability by keeping the design simple and readable while still experimenting with layout and visual contrast. I used a light grey background with high-contrast black and white elements to maintain clarity and accessibility. Even though I added creative features like the grid gallery, hover effects, and interactive black cat facts, I made sure everything stayed easy to navigate and consistent across all pages.

##What are you most proud of in your final project?
I’m most proud of the gallery layout and the interactive “black cat fact” feature. The gallery uses a structured grid system that combines images and text tiles to create a more dynamic visual experience. I also like the Myths vs Reality section because it not only looks visually distinct with contrasting colors, but also communicates a meaningful message about black cats and adoption awareness. The interactive fact button adds a small but engaging element that makes the site feel more alive.

##How do you see yourself using these skills later on?
I can see myself using these skills in future web design or creative projects, especially when building portfolios, personal websites, or interactive media pages. Learning how to structure multi-page sites, apply CSS Grid, and add simple JavaScript interactions gives me a strong foundation for more advanced design work. These skills are also useful for any field involving digital storytelling, design, or user experience.
