
#Reflection

##What’s something new you learned while building your site
While building this site, I learned how to use CSS Grid in a more intentional way to create structured layouts instead of just stacking elements. In my gallery page, I used a 3-column by 2-row grid that alternates between text blocks and images, which helped me understand how layout design can guide visual flow. I also improved my understanding of how to organize a multi-page website using consistent navigation and shared styling across pages.

##How did you balance creativity with usability?
I balanced creativity with usability by keeping the design simple and readable while still experimenting with layout and visual contrast. I used a light grey background with high-contrast black and white elements to maintain clarity and accessibility. Even though I added creative features like the grid gallery, hover effects, and interactive black cat facts, I made sure everything stayed easy to navigate and consistent across all pages.

##What are you most proud of in your final project?
I’m most proud of the gallery layout and the interactive “black cat fact” feature. The gallery uses a structured grid system that combines images and text tiles to create a more dynamic visual experience. I also like the Myths vs Reality section because it not only looks visually distinct with contrasting colors, but also communicates a meaningful message about black cats and adoption awareness. The interactive fact button adds a small but engaging element that makes the site feel more alive.

##How do you see yourself using these skills later on?
I can see myself using these skills in future web design or creative projects, especially when building portfolios, personal websites, or interactive media pages. Learning how to structure multi-page sites, apply CSS Grid, and add simple JavaScript interactions gives me a strong foundation for more advanced design work. These skills are also useful for any field involving digital storytelling, design, or user experience.



# Site Plan

## Home Page

**Goal:** Introduce black cats as a theme and guide users to explore different sections of the site

### Structure
- **Navigation:** Centered horizontal navigation bar with links to Home, History, Gallery, and Myths vs Reality
- **Hero:** Title “Black Cats” with a short tagline (“Mysterious. Misunderstood. Magical.”) and introductory paragraph explaining the purpose of the site
- **Lead-in:** Title “Black Cats” with a short tagline (“Mysterious. Misunderstood. Magical.”) and introductory paragraph explaining the purpose of the site
- **Main Content:** 
    -Three navigation cards (History, Gallery, Myths vs Reality)
    -Each card includes a short description and a link to the corresponding page
    -Cards use hover scaling for interactivity
- **Supporting Content:** none on this page
- **Call-to-Action:** Implicit CTA through “Learn More,” “View Gallery,” and “Explore” buttons within cards
- **Footer:** Simple footer with copyright text

## History Page

**Goal:** Inform users about the cultural and historical significance of black cats while correcting common misconceptions

### Structure
- **Navigation:**  Consistent header with site title and centered navigation links (Home, History, Gallery, Myths vs Reality)
- **Hero / Intro:**  Page heading “History of Black Cats” introducing the topic
- **Main Content:**  
    -Three paragraphs explaining:
        -Positive symbolism in ancient Egypt
        -Negative associations during the Middle Ages (witchcraft and superstition)
        -Modern perspectives and cultural differences (luck vs superstition)
    -External resource link for additional learning (opens in a new tab)
- **Supporting Content (optional):**  None
- **Call-to-Action:**  Subtle CTA through the external “Learn more” link and site navigation
- **Footer:**  Simple footer with copyright

## Myths vs Reality

**Goal:** Challenge common misconceptions about black cats while educating users through comparison and interaction

### Structure
- **Navigation:**  Consistent header with site title and centered navigation links (Home, History, Gallery, Myths vs Reality)
- **Hero / Intro:**  Page heading “Myths vs Reality” introducing the theme of debunking misinformation
- **Main Content:**  
    -Two-column grid layout 
    -Alternating “Myth” and “Reality” blocks:
        -Myth sections use a dark background to emphasize negativity and superstition
        -Reality sections use a light background to contrast with factual information
    -Hover effects subtly lift elements to add interactivity
- **Supporting Content (optional):**  
Adoption Section:
    -Informational text about adoption bias against black cats
    -Encourages users to reconsider perceptions
    -External link (“Find Adoptable Cats”) directing users to a real adoption site
- **Call-to-Action:**  
    -Direct CTA encouraging black cat adoption
    -Interactive feature (“Get a Black Cat Fact” button) that displays dynamic content using JavaScript
- **Footer:** Simple footer with copyright
